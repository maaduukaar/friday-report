# Управление Telegram-ботом Friday Report в фоновом режиме

В этой папке находится скрипт управления ботом: `manage.sh`. Он позволяет запускать бота в фоновом режиме, останавливать его, перезапускать и просматривать логи.

Все команды нужно выполнять из этой директории (`/home/ubuntu/.hermes/tools/friday-report`).

---

### 🟢 Проверить статус бота
Чтобы узнать, работает ли бот в данный момент:
```bash
./manage.sh status
```

### 🚀 Запустить бота в фоне
Если бот выключен и вы хотите его запустить:
```bash
./manage.sh start
```
Бот запустится в фоновом режиме и продолжит работать даже после того, как вы закроете терминал (сессию SSH).

### 🛑 Остановить бота
Чтобы выключить бота:
```bash
./manage.sh stop
```

### 🔄 Перезапустить бота
Если вы внесли изменения в код или конфигурацию и хотите применить их:
```bash
./manage.sh restart
```

### 📋 Просмотреть логи бота
Чтобы увидеть лог работы бота в реальном времени (нажмите **Ctrl+C** для выхода):
```bash
./manage.sh logs
```
Или просто откройте файл логов напрямую:
```bash
cat bot.log
```
