# 📊 Friday Report Automation (Трудозатраты ДИТ)

Скрипт для автоматизации заполнения еженедельного отчета о трудозатратах ДИТ в Яндекс Формах. 

Больше не нужно вручную вводить проценты в 20+ полей — скрипт сделает это за вас, проверит правильность суммы и пришлет скриншоты для подтверждения.

## ✨ Возможности

- **Автоматическое заполнение**: Проставляет все проценты по категориям.
- **Валидация суммы**: Скрипт не запустится, если сумма трудозатрат не равна 100%.
- **Автоматические скриншоты**: Делает скриншоты каждого шага и сохраняет их в папку `verified/` для контроля.
- **Поддержка динамической даты**: Автоматически выбирает текущий день в календаре.
- **Обработка ошибок**: Перезагружает страницу, если форма Яндекса временно недоступна.

## 🛠 Требования

- **Python 3.10+**
- **Playwright** (библиотека для автоматизации браузера)

## 🚀 Установка

1. Склонируйте репозиторий или скачайте файл `friday_report.py`.
2. Установите зависимости:
   ```bash
   pip install playwright
   ```
3. Установите браузеры для Playwright:
   ```bash
   playwright install chromium
   ```

## ⚙️ Настройка

1. Откройте `friday_report.py` в любом текстовом редакторе.
2. Вставьте ссылку на актуальную форму в переменную `URL` (примерно 51 строка):
   ```python
   URL = "https://forms.yandex.ru/u/..."
   ```
3. Настройте свои трудозатраты в блоке настроек в начале файла:

```python
# --- НАСТРОЙКИ ТРУДОЗАТРАТ (в сумме должно быть ровно 100) ---
SITES_ADMIN = 20         # Администрирование сайтов
SITES_DEVELOPMENT = 70   # Доработка сайтов support
REPORTING = 10           # Ведение (подготовка) отчетности
# ... и так далее
```

**Важно**: Сумма всех переменных должна быть строго равна **100**.

## 📖 Использование

Просто запустите скрипт:

```bash
python friday_report.py
```

После завершения в папке `verified/` появятся скриншоты:
1. `01_start_page.png` — Выбор подразделения и даты.
2. `02_name_page.png` — Выбор сотрудника.
3. `03_confirm_page.png` — Страница подтверждения.
4. `04_workload_filled.png` — Финальный вид заполненных полей перед отправкой.
5. `05_final_page.png` — Страница после нажатия кнопки «Отправить».
